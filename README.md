# sroi_v2

> **SZN version of sroi** —— 草莓采摘末端执行器 / 手持数据采集装置（第二版）

一套用于草莓采摘的**末端执行器**，同时具备两种使用形态：

- **手持数据采集形态** —— 由握把 + 板机式「录制触发按键」驱动，配合 RealSense D405 相机与空间定位 Tracker，用于采集遥操作 / 示教数据。
- **关节电机形态** —— 末端接口对接关节电机外法兰，由电机直接驱动，可挂载于机械臂执行实际采摘。

两种形态共用同一套平行夹爪 + 直线导轨的传动末端，仅更换「摇杆转接件」与「导轨座」即可切换。

## 机构原理

末端采用 **摇杆 + 连杆 + 直线导轨** 的传动方案，把单一旋转输入转换为夹爪的平行开合：

```
[输入]
 手持：按下 按钮(线性) ──▶ 核心块中心转轴上的 U 型凸轮槽 ──▶ 转轴(旋转)
 关节电机：电机轴 ──▶ 摇杆-转接电机 ──▶ 摇杆(旋转)
                                        │
                                        ▼
                              摇杆(旋转 5° → 50°)
                                        │
                                   连杆 ×2
                                        │
                                        ▼
                          转接块 ── MGN9C 滑块 ── 沿 MGN9 导轨平移
                                        │
                                        ▼
                                 平行夹爪 开 / 合

  松开时由弹簧提供回复力（夹爪自动张开）。
```

**关键传动参数（小开度配置）：**

| 参数 | 数值 |
|---|---|
| 摇杆旋转行程 | 45°（5° → 50°） |
| 对应滑块行程 | 24 mm |
| 当前导轨长度 | 156 mm（全开 ~96 mm） |
| 计划行程（草莓精细采摘） | ~40 mm（拟缩短导轨、缩小整体体积） |

> 设计取舍：圆弧形凸轮槽虽意图让传动更平顺，但实测会在中段产生「夹爪不动 → 冲击」的死区，反而更难控制；最终采用**直线的 U 型槽**，并在闭合段调整角度以放大夹持力——这是平顺性与夹持力之间的折中解。详见 [设计迭代记录](#设计迭代记录)。

## 装配指南

### 相机 / Tracker 总成

1. 将相机安装到**相机支架**：D405 背面两颗螺丝 → **2 × 沉头内六角 M3×8**
2. 将 Tracker 安装到 **Tracker 安装板**：中心螺丝 → **1 × 平头内六角 1/4″-20 × 5/16″**
3. 将「Tracker + 安装板」组合体转接到相机支架侧面 → **2 × 沉头内六角 M3×12**
   - 不需要 Tracker 时，组合体可整体拆下，相机支架仍保持小体积。
4. 将相机支架安装到**导轨座**（底部三孔） → **3 × 杯头内六角 M3×12**
5. 将树莓派盒子固定到**把手**（盒子侧面） → **2 × 沉头内六角 M3×10**

### 握把总成

1. 录制按键先安上**单通铜柱**。
2. 将录制触发按键固定到握把内部（需要较长的手），握把侧面两个沉孔 → **2 × 沉头内六角 M3×8**
3. 将电池的 **Type-C 母头**嵌入握把底盖预留空腔。
4. 两根导线穿过充电口配件预设的 U 型槽。
5. 通过充电口配件把 Type-C 母座严格约束在空腔内，并将配件固定到底盖 → **1 × 杯头内六角 M3×6**
6. 将电池塞入握把。
7. Type-C 公头（输出端）与录制按键的导线均从握把底盖的**过线孔**引出。
8. 将握把底盖安装到握把（底部） → **2 × 沉头内六角 M3×8**

### 树莓派盒子总成

1. 树莓派上 4 个固定通孔需先用 **Φ3 钻头**扩孔（原尺寸略小于 Φ3）。
2. 连接按键的 **3-pin 线**需提前焊接到树莓派针脚。
3. 将树莓派放入**盒子-底**，靠近 40-pin 排针的两个孔用于拧螺丝 → **2 × 杯头内六角 M3×6**（同时把树莓派与盒子-底固定）。
4. 盖上**墨水屏**。
5. 盖上盒子顶盖，从底部拧螺丝将顶盖锁在底座上 → **2 × 沉头内六角 M3×10**

> 紧固件头型、规格与数量见 [BOM.md](BOM.md)。

## 设计迭代记录

### 第一版 → 第二版（限位方式）
- **V1 问题**：夹爪滑动的限位完全靠打印件本身实现，使用中明显卡顿。
- **V2 尝试**：改用「光轴 + 铜套」约束（安装空间仅 10 mm，找不到这么小的直线轴承）。
- **V2 结果**：比纯 3D 打印更卡——滑动摩擦必须带间隙，不稳定；纯打印虽顺滑但有晃动，传动时仍会卡顿。

### 第二版 → 第三版（夹爪闭合 / 夹持力）
- **问题**：夹爪需带刀片剪断草莓茎；手持装置需能徒手剪下草莓，但拇指控制的按键无法向夹爪传导足够大的力（装置固有缺陷）。
- **改进 1（已实施）**：改动摇杆与连杆长度，在闭合段放大作用力——缩短摇杆（更大闭合力矩），加长连杆。
- **改进 2（暂缓，可选）**：当前 156 mm 导轨全开 96 mm，对草莓精细采摘过大（约需 40 mm 行程）。导轨改短 → 整体更小、更美观。

### 2026.7.3（周五）
- 夹爪微调，**尚未做刀片槽**；转接块微调；连杆加长；摇杆缩短以增大闭合力矩；初步调整摇杆凸轮槽位置。
- 计划：D405 相机支架再优化，把 Tracker 固定到相机后侧。

### 2026.7.4 凌晨
- **相机支架-D405 彻底优化**：只保留 D405 安装孔位，体积最小；侧面增加 Tracker 安装板接口；Tracker 可单独拆下。
- **Tracker 安装板完成**：先与 Tracker 组合（凸起约束姿态），再选择性转接到相机支架；Tracker 中心位于相机正后方，简化 TF 变换。
- **导轨座螺母孔优化**：3 处内嵌螺母孔改到「卡入后不掉落」的尺寸，方便安装。
- **树莓派盒子优化**：插口处尺寸优化；底盖下方加厚，落地更稳，避免墨水屏附近被暴力按坏。

### 2026.7.5
- **换用 MGN9 直线导轨后，丝般顺滑**——所有问题都源于传动不够顺滑。
- **运动参数测定**：小开度下，摇杆 5° → 50°（行程 45°），对应滑块行程 24 mm。
- **凸轮槽定型**：直线 U 型槽传动最平顺（圆弧槽会导致中段死区与冲击，效果差）；最终在 U 型槽闭合段调整角度以放大夹持力——平顺性与夹持力的折中解。
